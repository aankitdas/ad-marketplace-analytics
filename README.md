# Ad Marketplace Analytics Platform

End-to-end simulation and analytics platform for mobile advertising marketplaces.

## Features
- Marketplace auction simulation
- A/B experiment analysis with causal inference
- Churn prediction model
- Interactive dashboard

## Tech Stack
- Python, Pandas, NumPy
- XGBoost, scikit-learn
- Docker, Streamlit, Plotly
- Statistical analysis (scipy, statsmodels)
